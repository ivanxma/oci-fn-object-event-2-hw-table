Queue control model

Processing starts automatically when an enabled OCI Events rule delivers a mapped object event. There is no global start/stop switch in the queue because stopping a worker does not stop event intake.

To pause new intake, disable the applicable OCI rule. Use Details to retry or cancel an entry, edit one manual entry, or wake a worker for a specific binding.
